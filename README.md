# FOSS AI Browser

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)  
[![Android](https://img.shields.io/badge/Platform-Android-green.svg)]

A free, open-source, privacy-focused, and AI-powered web browser for Android. Inspired by FOSS browser projects and enhanced with intelligent AI features for smarter browsing.  

---

## Features

- **Open Source & Free:** Fully open-source under the MIT License.  
- **Privacy-Focused:** Blocks trackers, ads, and does not collect user data.  
- **Fast & Lightweight:** Built on Android WebView with a spoofed User-Agent for better compatibility.  
- **AI-Powered Features:**  
  - **AI Chat:** Ask questions and get answers directly inside the browser.  
  - **AI Explain:** Summarize articles, explain web content, or provide insights.  
- **Tabs:** Multiple tab support for convenient browsing.  
- **Dark Mode:** Built-in dark theme for comfortable night browsing.  
- **Download Manager:** Supports downloads with pause/resume.
